# Trader Behavior & Market Sentiment Analysis

## Objective
Analyze the relationship between Bitcoin market sentiment (Fear & Greed Index)
and trader performance using Hyperliquid historical data.

## Dataset
- Hyperliquid historical trade data
- Bitcoin Fear & Greed Index

## Approach
- Time-aligned millisecond trade data with daily sentiment data
- Focused on realized PnL for performance evaluation
- Engineered metrics such as win rate, trading volume, and net PnL
- Analyzed trader behavior across Fear and Greed regimes

## Key Insights
- Traders perform better during Fear regimes compared to Greed
- Greed phases show higher activity but lower profitability
- Evidence of contrarian traders profiting against sentiment extremes

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn
